# cultural-matching-app

Web application designed to help people organize shared cultural activities such as museum visits, exhibitions and events.

## Overview
The goal of the project is to connect users with similar cultural interests and help them organize group visits, improving the overall experience through social interaction.

## Main Features
- User profiles with cultural preferences and availability
- Matching between users with similar interests
- Exploration of museums and cultural events
- User authentication and data management
- Reviews and ratings of experiences
